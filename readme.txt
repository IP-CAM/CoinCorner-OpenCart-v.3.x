This is the official CoinCorner.com OpenCart Payment module for integration with Bitcoin

== Description ==

The CoinCorner OpenCart Payment module allows any site to accept payment in Bitcoin. Once connected via your API keys in your CoinCorner account, you are free to accept payments in Bitcoin and instantly be converted into GBP.

Features:

* Accept BTC payments
* Get the exact fiat amount back for your payments based on the currect price at CoinCorner.com
* Low 1% fees for accepting payments.
* Easy to use, easy to integrate, easy for your customers.

*Please note you will need an account at CoinCorner.com to use this plugin*

Once installed, your site will simply have a Pay with Bitcoin option and will be ready to go. Yay!


== Installation ==

To install, simply add this plugin to your OpenCart plugins and configure with your CoinCorner.com API keys.
1. Download https://www.coincorner.com/Content/Plugins/OpenCart-CoinCornerCheckout.ocmod.zip
2. Upload 'CoinCornerCheckout.zip' via the Admin panel > Extensions > Installer
3. Goto Extensions > Extensions & choose Payments. Click Install next to CoinCorner
4. In the same list next to CoinCorner click edit and add your API keys. If you do not have an account, you can sign up at CoinCorner.com
